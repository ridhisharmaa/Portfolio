Projects

ML Prediction Web App

A lightweight Flask-based web application that serves a trained machine learning model for real-time predictions. The goal was to move beyond notebooks and deploy a working ML system end-to-end.

Includes a trained ML model serialized as .pkl, an integrated preprocessing pipeline with StandardScaler, a simple HTML frontend, and a production deployment on AWS.

Tech Stack: Python, Flask, Scikit-learn, StandardScaler, AWS Elastic Beanstalk, EC2 (t3.micro), Nginx, Gunicorn

Live Demo GitHub Repo

Student Performance Prediction – Dockerized ML App

A fully containerized, production-oriented machine learning web application that predicts student performance from academic and demographic inputs.

Built with a custom preprocessing pipeline and serialized model artifacts, the system is packaged as a Docker image and deployed on AWS Elastic Beanstalk (Docker on Amazon Linux 2023) to simulate a real-world cloud deployment workflow.

Tech Stack: Python, Flask, Scikit-learn, Custom Pipeline, Docker, AWS Elastic Beanstalk, EC2, Gunicorn
